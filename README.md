# Luther College APC Mongo Admin Flask Web Application (LCAMAFWA)

This is a Flask web application that creates a graphical user interface for the MongoDB backend powering the Luther Course Proposal app. Essentially the user would be able to update the collections in the mongodb.

## Setup

1. python virtualenv --python=python3 path/to/name_of_virtualenv
2. Source path/to/name_of_virtualenv/bin/activate
1. pip install -r requirements.txt
2. python lc_apc_mongo_web_app.py

Some configurations may need to be corrected and those configurations can made in lc_apc_mongo_web_app.py.
